Heart Disease Analysis



Project ini merupakan tugas individu dari [MSIB]DS&AI course dari Startup Campus.



Dataset yang di gunakan pada analisi ini berasal dari Heart Disease Dataset on Kaggle



Analisis yang dilakukan mencakup berbagai model pembelajaran mesin dan teknik statistik untuk memperoleh wawasan dari Dataset, Berikut poin-poin utamanya:

    Data Understanding

    Identifying column


    • age: Age in years (ranging from 29 to 77)
    • sex: Gender (1 = male; 0 = female)
    • cp: Chest pain type (4 values)
    • trestbps: Resting blood pressure (in mm Hg on admission to the hospital)
    • chol: Serum cholesterol in mg/dl
    • fbs: Fasting blood sugar > 120 mg/dl (1 = true; 0 = false)
    • restecg: Resting electrocardiographic results (values 0, 1, 2)
    • thalach: Maximum heart rate achieved
    • exang: Exercise induced angina (1 = yes; 0 = no)
    • oldpeak: ST depression induced by exercise relative to rest
    • slope: The slope of the peak exercise ST segment
    • ca: Number of major vessels (0-3) colored by fluoroscopy
    • thal: Thalassemia (1 = normal; 2 = fixed defect; 3 = reversible defect)
    • target: Presence of heart disease (1 = yes; 0 = no)
  • Handling Outlier: Outlier are filtered using interquartile range.
  • Handling Imbalance Data: Oversampling data using SMOTE.
  • Feature Importance: Identified the most significant features influencing heart disease.
  • Model Selection: Different models like Logistic Regression, Decision Tree, Random Forest, Support Vector, XGBoost were used.
  • Model Optimization: Optimized models with the best hyperparameter combination using GridSearchCV.
  • Performance Metrics: Evaluated models using accuracy, precision, recall, and F1 score.

    Heart Disease Analysis Interactive Dashboard

    Feature Selection

    In the interactive dashboard, I used several features including:

    Metric

    • Number of Patients: Record Count

      • Show number of patients by number and percentage.

    Dimension

    • age

      • Using age filter to see patients age distributions.

    • cp

      • To see the number of patients who suffer from heart disease based on the type of chest pain with 0 = typical angina, 1 = atypical angina, 2 = non-anginal pain, 3 = asymptomatic.

    • trestbps

      • Show resting blood pressure per number of patients, add line for Systolic and Diastolic blood pressure.

      Systolic: Pressure exerted when blood is ejected into arteries, normal is 120mmHg or below.

      Diastolic: Pressure blood exerts within arteries between heartbeats, normal is 80mmHg or below.

    • thal

      • To see the number of patients who suffer from heart disease based on thalassemia with 0 = no, 1 = normal, 2 = fixed defect, 3 = reversible defect.

    • chol / target

      • Show serum cholesterol in mg/dl numbers by patients based on heart disease with target 0 = no, 1 = yes.

    • thalach

      • Show maximum heart rate achived numbers by patients based on heart disease with target 0 = no, 1 = yes.

    • sex

      • Sex distribution percetage based on number of patients and heart disease.
